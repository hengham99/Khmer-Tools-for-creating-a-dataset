# Kh_Drawing Tool – Initial Demo

<img width="92" height="87" alt="tool" src="https://github.com/user-attachments/assets/b009793f-9e34-4a55-9bcc-53797a2b7e07" />

We developed Kh_Drawing as a first-step tool for creating a local dataset on a Windows operating system. For this initial demo, the tool allows us to import data from a .txt file. Once imported, it opens words or characters manually, enabling users to load them directly from the text file.

At this stage, Kh_Drawing serves as a basic framework for dataset collection. However, the tool requires further updates to make it more flexible and efficient, supporting a smoother workflow and broader dataset management in future versions.
